# ffmpeg-mini-gui

A small application doing the usual for `ffmpeg`.  

![Image](https://github.com/user-attachments/assets/46465efc-f2d5-4c5c-8dce-0cdef12e4de1)

download the latest ffmpeg from [here](https://www.ffmpeg.org/download.html). Add the `ffmpeg.exe` near the appication.
